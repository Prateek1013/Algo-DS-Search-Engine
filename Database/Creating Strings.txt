
CSES - Creating Strings


Time limit: 1.00 s
Memory limit: 512 MB



Given a string, your task is to generate all different strings that can be created using its characters.

Input

The only input line has a string of length nn. Each character is between a–z.

Output

First print an integer kk: the number of strings. Then print kk lines: the strings in alphabetical order.

Constraints

1≤n≤81≤n≤8

Example

Input:
aabac

Output:
20

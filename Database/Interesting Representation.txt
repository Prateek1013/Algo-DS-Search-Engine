Given a positive integer NN, find two positive integers AA and BB, such that A−B=N and the number of distinct prime factors of A and B are equal. If there exist multiple pairs of values of A and B satisfying the given conditions, you can output any one of them.
Input Format

First line will contain T, number of testcases. Then the testcases follow. 
Each testcase contains of a single line of input, integer N. 

Output Format
For each testcase, output in a single line two space separated integers A and B.
If there are multiple answers, you can output any one of them.
Constraints

1≤T≤1000
1≤N≤1016
1≤A,B≤1018

Subtasks
Subtask 1 (100 points): Original constraints

                        Sample Input 1
                        





3
1
8
135



                        Sample Output 1
                        





3 2
97 89
1071 936


CSES - Number Grid


Time limit: 1.00 s
Memory limit: 512 MB



Consider a two-dimensional grid whose rows and columns are 11-indexed. Each square contains the smallest nonnegative integer that does not appear to the left on the same row or above on the same column.

Your task is to calculate the value at square (y,x)(y,x).

Input

The only input line contains two integers yy and xx.

Output

Print one integer: the value at square (y,x)(y,x).

Constraints

1≤y,x≤1091≤y,x≤109

Example

Input:
3 5

Output:
6

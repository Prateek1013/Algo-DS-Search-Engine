
There are 101101 citizens in Chefland. It is election time in Chefland and 33 parties, A,BA,B, and CC are contesting the elections. Party AA receives XAXA votes, party BB receives XBXB votes, and party CC receives XCXC votes.
The constitution of Chefland requires a particular party to receive a clear majority to form the government. A party is said to have a clear majority if it receives strictly greater than 5050 votes.
If any party has a clear majority, print the winning party (A, B or C). Otherwise, print NOTA.
Input Format

The first line of input contains a single integer TT, denoting the number of test cases. The description of TT test cases follows.
The first and only line of each test case contains 33 space-separated integers — XAXA, XBXB, and XCXC.

Output Format
For each test case, if any party has a clear majority, print the winning party (A, B or C). Otherwise, print NOTA.
You can print each letter of the string in any case (upper or lower) (for instance, strings Nota, nOtA and notA will be considered identical).
Constraints

1≤T≤5001≤T≤500
0≤XA,XB,XC≤1010≤XA,XB,XC≤101
XA+XB+XC=101XA+XB+XC=101


                        Sample Input 1
                        





3
80 19 2
20 55 26
50 1 50



                        Sample Output 1
                        





A
B
NOTA


Explanation
Test Case 11: Party AA has received 8080 votes, which is strictly greater than 5050.
Test Case 22: Party BB has received 5555 votes, which is strictly greater than 5050.
Test Case 33: None of the parties have received strictly more than 5050 votes.

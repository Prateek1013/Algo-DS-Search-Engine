
CSES - Reading Books


Time limit: 1.00 s
Memory limit: 512 MB



There are nn books, and Kotivalo and Justiina are going to read them all. For each book, you know the time it takes to read it.

They both read each book from beginning to end, and they cannot read a book at the same time. What is the minimum total time required?

Input

The first input line has an integer nn: the number of books.

The second line has nn integers t1,t2,…,tnt1,t2,…,tn: the time required to read each book.

Output

Print one integer: the minimum total time.

Constraints

1≤n≤2⋅1051≤n≤2⋅105
1≤ti≤1091≤ti≤109

Example

Input:
3
2 8 3

Output:
16


CSES - Permutations II


Time limit: 1.00 s
Memory limit: 512 MB



A permutation of integers 1,2,…,n1,2,…,n is called beautiful if there are no adjacent elements whose difference is 11.

Given nn, your task is to count the number of beautiful permutations.

Input

The only input line contains an integer nn.

Output

Print the number of beautiful permutations of 1,2,…,n1,2,…,n modulo 109+7109+7.

Constraints

1≤n≤10001≤n≤1000

Example

Input:
5

Output:
14

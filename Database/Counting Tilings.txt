
CSES - Counting Tilings


Time limit: 1.00 s
Memory limit: 512 MB



Your task is to count the number of ways you can fill an n×mn×m grid using 1×21×2 and 2×12×1 tiles.

Input

The only input line has two integers nn and mm.

Output

Print one integer: the number of ways modulo 109+7109+7.

Constraints

1≤n≤101≤n≤10
1≤m≤10001≤m≤1000

Example

Input:
4 7

Output:
781

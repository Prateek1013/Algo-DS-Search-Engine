
CSES - Money Sums


Time limit: 1.00 s
Memory limit: 512 MB



You have nn coins with certain values. Your task is to find all money sums you can create using these coins.

Input

The first input line has an integer nn: the number of coins.

The next line has nn integers x1,x2,…,xnx1,x2,…,xn: the values of the coins.

Output

First print an integer kk: the number of distinct money sums. After this, print all possible sums in increasing order.

Constraints

1≤n≤1001≤n≤100
1≤xi≤10001≤xi≤1000

Example

Input:
4
4 2 5 2

Output:
9
2 4 5 6 7 8 9 11 13


CSES - Maximum Subarray Sum II


Time limit: 1.00 s
Memory limit: 512 MB



Given an array of nn integers, your task is to find the maximum sum of values in a contiguous subarray with length between aa and bb.

Input

The first input line has three integers nn, aa and bb: the size of the array and the minimum and maximum subarray length.

The second line has nn integers x1,x2,…,xnx1,x2,…,xn: the array values.

Output

Print one integer: the maximum subarray sum.

Constraints

1≤n≤2⋅1051≤n≤2⋅105
1≤a≤b≤n1≤a≤b≤n
−109≤xi≤109−109≤xi≤109

Example

Input:
8 1 2
-1 3 -2 5 3 -5 2 2

Output:
8
